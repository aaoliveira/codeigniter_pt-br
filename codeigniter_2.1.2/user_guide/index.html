Bem vindo ao CodeIgniter

CodeIgniter é um Framework de Desenvolvimento de Aplicações - um toolkit - para pessoas que querem construir web sites usando PHP. Seu objetivo é permitir que você desenvolva projetos mais rapidamente do que você desenvolveria se estivesse escrevendo código a partir do zero, ao oferecer um rico conjunto de bibliotecas para as tarefas necessárias mais comuns, bem como uma interface simples e estrutura lógica para acessar essas bibliotecas. Mercante permite que você mantenha o foco em seu projeto minimizando a quantidade de código necessário para uma determinada tarefa.

Para Quem é o CodeIgniter?

O CodeIgniter é ideal para você se:

  • Você quer um framework com uma pequena curva de aprendizado.
  • Você precisa um desempenho excepcional.
  • Você precisa por ampla compatibilidade com as contas de hospedagem padrões que rodam uma variedade de versões e configurações de PHP.
  • Você quer um framework que requer uma configuração quase zero.
  • Você quer um framework que não requer que você use a linha de comando.
  • Você quer um framework que não exige lhe exige aderir a regras restritivas de codificação.
  • Você não quer ser forçado a aprender uma linguagem de templates (embora um parser de template está disponível como opção, se você desejar um).
  • Você evita a complexidade, favorecendo soluções simples.
  • Você precisa de documentação clara e completa.
