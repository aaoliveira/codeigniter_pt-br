Escrevendo Documentação

To help facilitate a consistent, easy-to-read documentation style for CodeIgniter projects, EllisLab is making the markup and CSS from the CodeIgniter user guide freely available to the community for their use. For your convenience, a template file has been created that includes the primary blocks of markup used with brief samples.

Para ajudar a facilitar maneira consistente e fácil no estilo de ler a documentação para os projetos codeIgniter, EllisLab está tornando a marcação CSS a partir do guia do usuário do CodeIgniter livremente à disposição da comunidade para a sua utilização. Para sua conveniência, um arquivo que inclui os blocos primários de marcação utilizados com amostras breves foi incluído.
