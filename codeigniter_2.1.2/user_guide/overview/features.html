Características do CodeIgniter

Features in and of themselves are a very poor way to judge an application since they tell you nothing about the user experience, or how intuitively or intelligently it is designed. Features don't reveal anything about the quality of the code, or the performance, or the attention to detail, or security practices. The only way to really judge an app is to try it and get to know the code. Installing CodeIgniter is child's play so we encourage you to do just that. In the mean time here's a list of CodeIgniter's main features.

As características em si são uma forma muito pobre para julgar um aplicativo, uma vez que não lhe dizem nada sobre a experiência do usuário, ou o quão intuitivo ou inteligente foi concebido. As características não revelam nada sobre a qualidade do código, o desempenho, a atenção ao detalhe ou práticas de segurança. A única maneira de realmente julgar uma aplicação é testá-la e conhecer o código. Instalar o CodeIgniter é brincadeira de criança, por isso recomendamos que você faça exatamente isso. Nesse meio tempo, aqui está uma lista das principais características do CodeIgniter.

  • Sistema Baseado em Model-View-Controller
  • Extremamente Leve
  • Classes de Banco de Dados Cheias de Recursos com Suporte para Várias Plataformas
  • Suporte a Banco de Dados com Active Record
  • Validação de Formulários e Dados
  • Filtragem de Segurança e XSS
  • Gerenciamento de Sessões
  • classe de Envio de E-mail. Suporta Anexos, E-mail em HTML/TEXTO, múltiplos protocolos (sendmail, SMT e MAIL) e mais.
  • Biblioteca de Manipulação de Imagem (corte, redimensionamento, rotação, etc.). Suporte à GD, ImageMagick e NetPBM
  • Classe de Upload de Arquivo
  • Classe de FTP
  • Internacionalização
  • Paginação
  • Criptografia de Dados
  • Benchmarking
  • Cache Total de Página
  • Log de Erros
  • Perfil de Aplicação
  • Classe de Calendário
  • Classe de User Agent
  • Classe de Compactação Zip
  • Classe de Motor de Template
  • Classe de Trackback
  • Biblioteca XML-RPC
  • Classe de Teste Unitário
  • URLs Amigáveis
  • Roteamento Flexível de URI
  • Suporte para Extender Hooks e Classes
  • Ampla Biblioteca de "ajuda" das Funções
