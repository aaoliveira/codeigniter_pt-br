Iniciando com o CodeIgniter

Qualquer aplicativo de software exige algum esforço para o aprendizado. Nós fizemos o nosso melhor para minimizar a curva de aprendizado, tornando o processo o mais agradável possível.

O primeiro passo é a instalação do CodeIgniter, então, leia todos os tópicos da seção Introdução localizados na Tabela de Conteúdo.

A seguir, leia cada uma das páginas de Tópicos Gerais na ordem que são apresentadas. Cada tópico se baseia no anterior e inclui exemplos de código que você deve tentar fazer.

Depis de entender os princípios básicos, você estará pronto para explorar as páginas de Referências das Classes e Referência dos Helpers para aprender a utilizar as bibliotecas e helpers nativos.

Sinta-se à vontade para participar de nosso Fórum da Comunidade se tiver alguma dúvida ou problema e nosso Wiki para ver exemplos de códigos postados por outros usuários.
